Trait Polarity

pub trait Polarity: 'static + Sealed { }

An empty trait used to denote Polarity: either send or receive.

SendPort and RecvPort have identical representations (via Port) but are not interchangable, so SEND and RECV which implement this trait are used to differentiate between the two polarities.
